---
title: "Claude Code"
---

Pioneer supports a growing list of coding agents. Point your agent at Pioneer's inference endpoint and switch between models using each agent's native `/model` command.

Steps to integrate Claude Code with Pioneer:

1. Download and [set up](https://code.claude.com/docs/en/quickstart) Claude Code.
2. Open `~/.zshrc` and add these two lines at the bottom to point Claude Code to Pioneer:
   ```shellscript
   export ANTHROPIC_AUTH_TOKEN="<Your Pioneer API key>"
   export ANTHROPIC_BASE_URL="https://api.pioneer.ai/"
   ```
3. Then either open a new terminal or run `source ~/.zshrc` in existing ones.
4. Start Claude Code
   ```shellscript
   claude
   ```
5. Switch between Pioneer models with `/model` command.
